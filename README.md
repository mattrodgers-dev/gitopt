# GitOpt

Simplify Git workflows through intelligent automation.

This will be a CLI tool + web dashboard that analyzes Git repositories, identifies workflow inefficiencies, and automates cleanup tasks.

## Tech Stack

- **Language:** Python
- **CLI:** Click
- **Git Integration:** GitPython
- **Terminal Formatting:** Rich
- **Database:** SQLAlchemy + SQLite
- **Web API:** FastAPI + uvicorn
- **Frontend:** React
- **Testing:** pytest
